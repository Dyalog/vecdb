# README #

Current version: 0.1.1

### What is this repository for? ###
`vecdb` is a simple "columnar database" based on memory-mapped files, written in and for Dyalog APL.

### How do I get set up? ###

Clone/Fork the repo, and

    ]load vecdb.dyalog

### Tests ###
    ]load TestVecdb.dyalog
    #.TestVecdb.Run
